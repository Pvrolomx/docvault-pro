# DocVault Pro

Sistema de gestion documental para clientes.

## Features
- Gestion de multiples clientes
- Documentos personales con vencimiento (Pasaporte, FM3, etc)
- Documentos legales multipagina (Escrituras, Predial, etc)
- Alertas de vencimiento
- PWA instalable

## Stack
- Next.js 14
- localStorage
- PWA
